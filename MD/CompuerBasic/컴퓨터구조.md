컴퓨터 동작 원리  
0x => 16진수 앞에 붙는 숫자(원칙)  
16진수를 쓰는 이유 = 2진수를 4개씩 묶으면 16진수로 표현 가능  
Encoding = 컴퓨터가 <-> 사람이 이해할 수 있는 정보를 가져옴  
32bit => RAM 4GB 의 의미 = 한번에 2^32 크기만 전달할 수 있다  
64bit => 2^64 bit == 192 GB
32비트 정수의 범위 = -2^31 ~ 2^31 -1 (1을 빼는 이유는 0때문, 31제곱은 맨앞의 비트는 부호를 나타내기 때문)
